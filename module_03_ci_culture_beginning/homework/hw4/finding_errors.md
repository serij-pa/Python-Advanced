## Отчет по тестированию класса Person
1. Ошибка: NameError
   * Место ошибки : функция get_age()
   * Меры по устранению : импорт datetime.datetime  
<br/>
2. Ошибка: AssertionError
   * Место ошибки : функция get_age()
   * Меры по устранению : 
   необходимо поменять местами 
   <i><b>self.yob</b></i> и <i><b>now.year</b></i>  
<br/>
3. Ошибка : не присваивается новое имя
   * Место ошибки : функция set_name()
   * Меры по устранению : необходимо аргументу 
   self.name присвоить аргумент функции set_name()  
<br/>
4. Ошибка : не присваивается новый адресс
   * Место ошибки : функция set_address()
   * Меры по устранению : необходимо аргументу 
   self.address присвоить аргумент функции set_address()
